Detailed Description

Armazena e processa o jogo em si.

Define a sequência lógica, por meio de métodos específicos, para que o truco seja desenvolvido.

◆ checaPedidoTrucoBot1()

void Jogo::checaPedidoTrucoBot1 ( int  jaPediu)

Método para avaliar pedido de truco do bot 1.

Implementa a lógica para armazenar a resposta dos outros jogadores em relação ao pedido de truco do bot1. Além disso, caso a checagem de truco seja realizada, mas a resposta foi dada por algum valor diferente de 1 ou 2, ele indica um erro e repete a checagem novamente.

Parameters
Recebeum número inteiro como parâmetro para identificar se a checagem já foi realizada, mas estava com erro nas respostas dos outros jogadores.

Função com polimorfismo para jogar a carta.

Implementa a lógica para que, a partir dos parâmetros recebidos, seja possível identificar qual tipo de jogador vai lançar a carta. Sendo assim, caso seja um bot será utilizado o método jogarCarta do bot com o respectivo índice da carta, caso seja um humano o método utilizado será do usuário.

Parameters
Recebecomo parâmetro o endereço de um jogador e a carta que será jogada.
Returns
Retorna a carta que o usuário selecionou.
