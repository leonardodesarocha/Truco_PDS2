Detailed Description

Cria dentro do jogo um sistema de conta dos jogadores.

e armazena os dados para elaborar o ranking.

Implementa a lógica de logar dentro do jogo com uma conta que já existente ou cadastrar um novo usuário. Além disso, também armazena informações das partidas anteriores para elaborar um ranking entre os jogadores.
