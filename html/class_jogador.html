Detailed Description

Armazena e processa as informações do jogador.

Implementa a lógica para definir as informações básicas dos jogadores, como por exemplo, o nome, quais cartas ele tem acesso, se ele deseja pedir truco e qual carta ele deseja jogar.

Lógica para selecionar uma das três cartas disponíveis para o jogador.

De acordo com o parâmetro recebido, é selecionada a carta que o jogador deseja jogar e, consequentemente, a mesma carta é deletada do conjunto de cartas do jogador.

Parameters
Númerointeiro entre 1 e 3 para identificar a carta selecionada.
Returns
Retorna a carta selecionada do tipo Carta.
